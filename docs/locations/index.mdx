# Locations

API Muslim v2 provides a way to fetch all locations available to be passed as a parameter to the prayer times endpoints. This includes cities (Indonesia: kota) and regencies (Indonesia: kabupaten).

:::info

"HTTP GET" refers to the HTTP method: GET.
Feel free to read more about it in the [HTTP Methods](https://developer.mozilla.org/en-US/docs/Web/HTTP/Methods) documentation.

:::

```http title="HTTP GET"
https://api.myquran.com/v2/sholat/kota/semua
```

Performing a request to this endpoint will return something like this:

:::info

The elipsis (`...`) indicates that there are more items in the list, but they are not shown here for brevity.

:::

```json title="Response in JSON format"
{
	"status": true,
	"request": {
		"path": "/sholat/kota/semua"
	},
	"data": [
		{
			"id": "1001",
			"lokasi": "KAB. LAMPUNG TENGAH"
		},
		{
			"id": "1002",
			"lokasi": "KAB. LAMPUNG UTARA"
		},
		{
			"id": "1003",
			"lokasi": "KAB. LAMPUNG SELATAN"
		},
        ...
	]
}
```
