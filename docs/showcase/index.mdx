## iMasjid

![iMasjid](https://raw.githubusercontent.com/myquran-com/iMasjid/refs/heads/main/preview.png)

iMasjid is a web based application that provides prayer times information with customizable contents, from the mosque name, address, all the way to location as the base for the prayer times calculation. With the simple and intuitive interface, iMasjid is designed to be displayed on a TV screen in the mosque, providing the congregation with accurate prayer times and other relevant information.

## Lintar Adzan Indonesia

Islamic prayer times application for Android that features:

1. Daily and monthly prayer times for all cities in Indonesia.
2. Qibla direction and compass using Google Qibla Finder.
3. Google Maps/Location integration.
4. Notifications for prayer times.

### Tech Stack

- Lazarus
- Lazarus Android Module Wizard
