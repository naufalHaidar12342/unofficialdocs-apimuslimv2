# Get Quran Verses

Here are some examples of how to get Quran verses using the API.

:::info
In API Muslim v2, the developers put Indonesian as the main focus since it is intended for Indonesian users. Therefore, you will find lots of Indonesian words in the API responses for getting Quran verses.
:::

## Get Random Quran's Verses

For getting random verses, API Muslim v2 accept `acak` and `random` parameter.

### Get random verse using `acak` parameter

:::info

`acak` is Indonesian for "random"

:::

```http title="HTTP GET"
https://api.myquran.com/v2/quran/ayat/acak
```

## Get Specific Quran's Verse Using Surah and Ayat Number

API Muslim v2 use numbers to identify Surah (Chapter) and Ayat (Verse) in the Quran. We can use the following endpoint to get a specific verse:

```http title="HTTP GET"
https://api.myquran.com/v2/quran/ayat/{surat-number}/{ayat-number}
```

In this example, we will use the first Surah Al-Fatihah (The Opening) and the first Ayat (Verse) in it.

```http title="Get Surah Al-Fatihah, Ayat 1"
https://api.myquran.com/v2/quran/ayat/1/1
```
